Vue Router基础

安装

  • 直接下载 / CDN
    点我下载

  • NPM

    npm install vue-router

    注：
    如果在一个模块化工程中使用它，必须要通过 Vue.use()明确地安装路由功能：

import Vue from 'vue'       
import VueRouter from 'vue-router'

Vue.use(VueRouter)

起步

在开始之前可以先了解前端路由的两种模式：

  • hash模式和history模式

动态路由匹配

举例说明

假如现在有一功能需求：父组件放置登录/注册按钮，点击相应的按钮，显示相应的子组件。如下图所示：

注： 简单来说，实现此功能，并不需要父子组件，用v-if控制固定部分显示隐藏，即可实现，但为了体现组件化开发，在此用父子组件完成此功能

实现步骤

  1. 定义login组件和register组件

     //login组件模板对象
 var login={
     template:'<h1>登录组件</h1>'
 }
 //register组件模板对象
 var register={
     template:'<h1>注册组件</h1>'
 }

  2. 引用vue-router.js文件(在此使用直接下载方式)

    <script src="vue-router.js"></script>

注意：引用vue-router.js文件之前必须先引用vue.js文件

  1. 创建一个路由对象

注： 当导入vue-router包之后，在window全局对象中，就有了一个路由的构造函数，叫做VueRouter，在new路由对象的时候，可以为构造函数，传递一个配置对象

var routerObj=new VueRouter({

})

配置对象：

  • routes：表示路由匹配规则,每个路由规则，都是一个对象，这个规则对象身上，有两个必须属性：【重点！！！】
    • 属性1(必须)：path—监听哪个路由链接地址
    • 属性2(必须)：component—如果路由是前面匹配到的path,则展示component属性对应的那个组件，component属性值，必须是一个组件的模板对象
    • 属性3(选填)：redirect—默认路由
    • 属性4(选填)：children—实现路由嵌套
  • linkActiveClass：表示当前路由选中样式css类名
var routerObj=new VueRouter({
     routes:[
         /* 如果当前路径是 / 默认跳转login界面，符合功能需求*/
         { path:'/',redirect:'/login' },
         { path:'/login',component:login },
         { path:'/register',component:register}
     ],
     /* 修改当前路由选中样式css类名为：myactive，方便后期设置样式 */
     linkActiveClass:'myactive'
})

  1. 将路由规则对象注册到vm实例上，用来监听URL地址的变化，然后展示对应组件

    var vm=new Vue({
 el:'#app',
 data:{},
 methods:{},
 router:routerObj
});

  2. 在界面放置链接和占位符

  • 链接router-link:router-link默认渲染一个a标签
  • 占位符，放置子组件router-view:vue-router提供的元素，当做占位符
    <div id="app">
<!--链接-->
<router-link to="/login">登录</router-link>
  <router-link to="/register">注册</router-link>
  <!--占位符，放置子组件-->
  <!--transition  Vue提供的动画属性-->
  <transition mode="out-in">
      <router-view></router-view>
  </transition>
</div>
  1. 设置css样式(选中当前路由样式和子组件切换动画)
    <style>
 /* 当前路由样式 */
 .myactive{
     color: red;
     font-weight: 800;
     font-style: italic;
     text-decoration: underline;
 }
 /* 子组件切换动画 */
 .v-enter,
 .v-leave-to{
     opacity: 0;
     transform: translateX(140px);
 }
 .v-enter-active,
 .v-leave-active{
     transition: all 0.5s ease;
 }
</style>

    路由传参

    如果需要传递参数，vue-router提供两种方式
  • $route.query方式
  • $route.params方式

$route.query方式

  • 类型：Object
  • 一个 key/value 对象，表示 URL 查询参数
    假设要为刚才的login组件传递参数，如：id=1&name="张三"
    <router-link to="/login?id=1&name=张三">登录</router-link>

在login组件实例上应该接收传递的参数

var login={
    template:'<h1>登录----{{$route.query.id}}----{{$route.query.name}}</h1>',
    data(){
        return{
        }
    }
}

$route.params方式

  • 类型：Object
  • 一个 key/value 对象，包含了动态片段和全匹配片段
    假设要为刚才的login组件传递参数，如：id=1&name="张三"
    <router-link to="/login/1/张三">登录</router-link>

修改路由规则属性routes

routes:[
    // 对应参数
    { path:'/login/:id/:name',component:login }
]

在login组件实例上应该接收传递的参数

var login={
    template:'<h1>登录---{{$route.params.id}}---{{$route.params.name}}</h1>',
    data(){
        return{
        }
    }
}

实现效果

路由嵌套

使用路由配置对象children属性，实现路由嵌套

举例说明

在刚才的login组件和register组件外部再放置一个父组件account也就是说：account有两个子孩子—login组件和register组件

实现步骤

  1. 定义account组件
    /* 这里采用在外部定义模板方式 */
var account={
 template:'#tmp1'
}
var login={
 template:'<h1>我是登录</h1>'
}
var register={
 template:'<h1>我是注册</h1>'
}
<template id="tmp1">
    <div>
        <h1>这是Account组件</h1>
        <!--链接-->
        <router-link to="/account/login">登录</router-link>
        <router-link to="/account/register">注册</router-link>
        <!--占位符-->
        <router-view></router-view>
    </div>
</template>
  1. 创建路由对象，进行路由配置

注：使用children属性，实现嵌套路由，同时，子路由的path前面，不要带/ ，否则将永远以根路径开始请求

var router=new VueRouter({
    routes:[
        { 
            path:'/account',
            component:account,
            //使用children属性，实现子路由，同时，子路由的path前面，不要带/ ，否则将永远以根路径开始请求
            children:[
                { path:'login',component:login },
                { path:'register',component:register }
            ]
        }
    ]
})
  1. 将路由规则对象注册到vm实例上，用来监听URL地址的变化，然后展示对应组件
    var vm=new Vue({
 el:'#app',
 data:{},
 methods:{},
 /* router:router===router */
 router
});

命名视图

有时候想同时 (同级) 展示多个视图，而不是嵌套展示，例如创建一个布局，有 header(头部) sidebar (侧导航) 和 main (主内容) 三个视图，这个时候命名视图就派上用场了。你可以在界面中拥有多个单独命名的视图，而不是只有一个单独的出口。如果 router-view 没有设置名字，那么默认为 default

实现步骤

  1. 定义header、sidebar、main组件

    var header={
 template:'<h1 class="header">头部区域</h1>'
}
var sidebar={
 template:'<h1 class="left">侧边栏</h1>'
}
var main={
 template:'<h1 class="main">主体区域</h1>'
}

  2. 创建路由对象，进行路由配置

使用路由配置属性components，实现命名视图

var router=new VueRouter({
    routes:[
        { 
            path:'/',components:{
                'default':header,
                'sidebar':sidebar,
                'main':main
            }
        }
    ]
})

  1. 将路由规则对象注册到vm实例上，用来监听URL地址的变化，然后展示对应组件

    var vm=new Vue({
 el:'#app',
 data:{},
 methods:{},
 router
});

  2. 在界面放置组件
    router-viewname属性用来放置对应组件

<div id="app">
    <!--header-->
    <router-view></router-view>
    <div class="container">
        <!--sidebar-->
        <router-view name="sidebar"></router-view>
        <!--main-->
        <router-view name="main"></router-view>
    </div>
</div>
  1. 样式
    <style>
 *{
     margin: 0;
     padding: 0;
 }
 .header{
     background: orange;
     height: 80px;
 }
 .container{
     display: flex;
     height: 600px;
 }
 h1{
     margin: 0;
     padding: 0;
 }
 .left{
     background: lightgreen;
     flex:2;
 }
 .main{
     background: lightpink;
     flex: 8;
 }
</style>
