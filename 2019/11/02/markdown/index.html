markdown是什么？

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面，Markdown文件的后缀名便是".md"。

简单来说：Markdown用于纯文本格式编写文档->HTML页面
试想一下？手敲纯文本格式文档就可以展示HTML页面岂不是很爽~

markdown优点

  • 语法简单，能快速编写，几分钟就能了解它的全部语法
  • 能直接阅读，而没有HTML的各种标签的干扰

语法详解

标题

使用n个#加一个空格表示n级标题。

# 标题H1
## 标题H2
### 标题H3
#### 标题H4
##### 标题H5

而在预览时，解释器自动帮我们解释成HTML语法，如下：

<h1>标题H1</h1>
<h2>标题H2</h2>
<h3>标题H3</h3>
<h4>标题H4</h4>
<h5>标题H5</h5>

强调

星号或者是一个下划线包起来。

粗体

**粗体**
__粗体__

效果：
粗体
粗体

斜体

*斜体*
***粗斜体***

效果：
斜体
粗斜体

引用文本

区块引用使用>表示。

> 1.引用文本 Blockquotes
> 2.引用的行内混合 Blockquotes
> 3.这是另一个区块引用
>> 这是一个二级嵌套引用
>>> 这是一个三级嵌套引用

效果：

1.引用文本 Blockquotes
2.引用：如果想要插入空白换行即<br />标签，在插入处先键入两个以上的空格然后回车即可，普通链接。
3. 这是另一个区块引用

这是一个二级嵌套引用

这是一个三级嵌套引用

锚点与链接

格式是：链接的文字放在[]中，链接地址放在随后的（）中。

1.[普通链接](https://www.baidu.com/)
2.[普通链接带标题](https://www.baidu.com/ "普通链接带标题")
3.直接链接：
4.[mailto:test.test@email.com](mailto:test.test@email.com)

效果
1.普通链接
2.普通链接带标题
3.直接链接：https://www.mdeditor.com
4.mailto:test.test@email.com

图片

格式是：链接的文字放在![]中，链接地址放在随后的（）中。!必须

列表

- 列表一
- 列表二
- 列表三

* 列表二
* 列表三

  • 列表一
  • 列表二

+ 列表一
+ 列表二
    + 列表二-1
    + 列表二-2
    + 列表二-3
+ 列表三
    * 列表一
    * 列表二

1. 第一行

表格

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

对齐方式

我们可以设置表格的对齐方式：

  • -:设置内容和标题栏居右对齐。
  • :-设置内容和标题栏居右对齐。
  • :-:设置内容和标题栏居右对齐。
左对齐 右对齐 居中对齐
单元格 单元格 单元格
单元格 单元格 单元格

代码块

单行代码

如果代码量比较少，只有单行的话，可以用单反引号包起来。反引号是键盘esc键下方的那个按键。

`
这是一个代码块
`

效果：
<p>这是一个代码块</p>

多行代码

可以使用三个反引号来引用代码块。也可以使用一个tab或者是四个空格来表示 。在上方的```后可以添加代码语言，可以对关键字进行不同颜色的显示。

```代码语言
内容
```

  function test() {

转义

转义就是将一些特殊字符转换成正常显示的样子，和大多数编程语言相同使用反斜杠()表示。下面简单列举几个。

* \\
* \`
* \*
* \!

效果：

  • \
  • `
  • *
  • !

分割线

分割线可以由* - _（星号，减号，底线）这3个符号的至少3个符号表示，注意至少要3个，且不需要连续，有空格也可以,但是两个符号之间最多只能有一个空格。建议使用减号-表示

---
_ _ _
* **
